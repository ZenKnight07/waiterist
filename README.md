# Waiter Assistant

A offline **Waiter - Order Management & Kitchen Order Ticket (KOT) system** built with pure HTML, CSS, and JavaScript.

Designed for restaurants and cafes to manage tables, take orders, and generate printable kitchen tickets in a fast interface.

---

## Features

### 1. Table Management
- 14-table interactive dashboard
- Real-time table status:
  - Vacant
  - Ordering
  - Active with items
- Persistent data

### 2. Order System
- Category-based digital menu:
- Add / remove item quantities instantly
- Live order updates per table
- Custom kitchen notes per table

### 3. KOT (Kitchen Order Ticket)
- Auto-generated thermal-style KOT
- Printable using native browser print
- Clean receipt format for kitchen use
- Copy raw KOT text to clipboard

### 4. Retro Cyberpunk UI
- Neon-themed retro terminal design
- Scanline effects and glow animations
- Fully responsive layout
- Smooth interactive transitions

### 5. Offline Support
- No backend required
- Fully client-side application
- Works directly in browser

---

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
- Browser Print API

---

## How to Use
- Open the app in a browser
- Select a table (T-1 to T-14)
- Choose food category
- Add items to the order
- Add optional kitchen notes
- Click Finalize / KOT
- Print or copy the kitchen ticket
